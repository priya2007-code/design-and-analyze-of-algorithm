# Interpolation Search Analysis

## Overview
This project implements the Interpolation Search algorithm and compares its performance with Binary Search.

## Features
- Interpolation Search
- Binary Search
- Performance Benchmarking
- Comparison Count Analysis

## Time Complexity
Interpolation Search:
- Average: O(log log n)
- Worst: O(n)

Binary Search:
- Average: O(log n)
- Worst: O(log n)

## Run

python interpolation_search.py
